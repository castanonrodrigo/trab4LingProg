Descricao Geral:
  *O programa consiste em um catalogo de pacientes implementado atraves de uma arvore binaria
  *O usuario pode escolher entre montar um catalogo de dois tipos diferentes de paciente: pediatraia ou oncologia
  *Sao permitidas as seguintes operacoes:insercao, busca e visualizacao
  *apos a selecao de um tipo de paciente, todas as operacoes serao realizadas para o mesmo. Para criar um catalogo
  do outro tipo, eh necessario reiniciar o programa

Gerar Executavel:
  *Para gerar o executavel do programa basta rodar o comando "make" na pasta do projeto

Guia de uso:
  *somente numeros devem ser utilizados para a navegacao nos menus
  *o programa nao suporta a inclusao de nomes compostos ou nome e sobrenome
  *apos a execucao de um comando do menu, eh necessario apertar um numero qualquer(diferente de -1) para voltar ao menu
  *no primeiro menu o usuario seleciona para qual tipo de paciente deseja criar um catalogo
  *todas as operacoes realizadas apos a escolha do tipo de paciente serao realizadas para esse tipo
  *no segundo menu o usuario seleciona qual operacao ira realizar
  *apos a selecao no segundo menu, aparecerao instrucoes de acordo
